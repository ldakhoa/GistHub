## Installation

### Requirements
- Xcode 14.1
- iOS 16.1
- SwiftLint: `brew install swiftlint`

To get started, run these instructions:
```
git clone https://github.com/ldakhoa/GistHub
cd GistHub
open GistHub.xcodeproj
```
and setup project with these [instructions](./CONTRIBUTING.md).
